# Intelligent Customer Churn Prevention System

> ML-powered customer churn prediction with AI-generated retention strategies

**Status:** In Development | **Started:** December 2024

## Project Overview

This project builds an end-to-end machine learning system that:

- Predicts customer churn risk with 85%+ accuracy
- Generates personalized retention strategies using AI
- Provides ROI-focused recommendations for retention campaigns
- Demonstrates business value through interactive dashboards

## Business Goal

Help telecom companies reduce customer churn and optimize retention spending by:

- Identifying at-risk customers 60 days in advance
- Creating cost-effective, personalized retention strategies
- Achieving 4.5x ROI on retention campaigns

## Tech Stack

- **ML:** Python, scikit-learn, XGBoost, SHAP
- **GenAI:** OpenAI GPT-4
- **Data:** pandas, numpy
- **Viz:** Plotly, Streamlit
- **MLOps:** Git, pytest

## Dataset

**Telco Customer Churn Dataset** (IBM Sample Data)

- 7,043 customers
- 21 features (demographics, services, billing)
- Target: Churn (Yes/No)
